1. 群，子群，陪集

Definition 1.1 群 Groups

一个非空集合G及一个二元运算 $\cdot $ 可能具有以下五种性质：
$\forall a,b,c\in G$

  1. 封闭性: $a\cdot b\in G$
  2. 结合性: $a\cdot (b\cdot c) = (a\cdot b)\cdot c$
  3. 单位元: $\exists e\in G，a\cdot e = e\cdot a = a$
  4. 逆元素: $\exists b\in G，a\cdot b = b\cdot a = e$ (b称为a的逆元素，记为$a^{-1}$)
  5. 交换性: $a\cdot b = b\cdot a$

当满足性质1, 2, 3, 4时，称(G,$\cdot $)为一个群 group；
当满足性质1, 2时，称(G,$\cdot $)为一个幺半群 monoid；
当满足性质1, 2, 3, 4, 5时，称(G,$\cdot $)为一个阿贝尔群 abelian group；

Example 1.1

  1. 整数集Z = {…,-n,…,-1,0,1,…,n…} 是单位元为0的加法阿贝尔群，集合Z* = Z - {0} 由于不满足逆元素不是乘法群；
  2. 有理数集Q 是单位元为0的加法阿贝尔群，集合Q* = Q - {0} 是单位元为1的乘法阿贝尔群；
